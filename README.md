# Simple Youtube Clone React Application

## Steps 
    Open App.js in Src folder and add
    
    your Youtube API Key
    
    get it from  https://console.developers.google.com
    
     paste it in line 33 in App.js
     
## Run the app
  1. Open youtube-clone folder
  
  2.type 
  
      yarn install
      yarn start
